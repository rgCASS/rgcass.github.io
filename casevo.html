Casevo

认知主体与社会演化模拟器

基于多智能体的社会传播模拟器

GitHub 开源地址

专为科研设计的开源AI科研助手，其核心是一个多智能体模拟框架，通过集成大语言模型构建具备高级认知能力的智能体，支持从场景设置到复杂社会网络建模的完整仿真框架。

核心特点

用于生成和分析模拟社会行为的生成式研究工具

LLM驱动的认知智能体

项目的基石是利用大语言模型作为模拟中每个智能体的"大脑"。这使得智能体能够进行推理、理解语境并生成复杂的行为，超越了传统基于简单规则的智能体。

动态社会建模

该框架专为模拟动态社会过程而设计，能够有效支持社会网络分析、舆情动态演化、复杂社会系统中的行为预测等多种研究场景。

高保真度与灵活性

通过引入具备高级认知能力的智能体，Casevo显著提升了模拟的真实性和灵活性，能够促成"更真实和灵活的智能体互动"。

可扩展与可定制的框架

Casevo基于成熟的Python库Mesa构建，继承了其模块化和灵活性。研究人员可以高度定制智能体的角色、行为、记忆以及整个模拟环境，以适应广泛的研究问题。

主要功能

全方位支持社会模拟研究

社会交互模拟

模拟复杂的社会交互过程，如选举辩论、公共舆论传播等，再现个体之间的互动和信息交流。

动态社会网络建模

支持构建和动态调整社会网络结构，反映个体关系的演变，适用于信息传播、社交影响等研究场景。

个体行为与决策模拟

基于链式思维（CoT）和检索增强生成（RAG）等机制，代理能进行多步推理和基于历史记忆的决策，模拟个体在复杂情境下的行为选择。

大规模并行处理

具备并行优化模块，高效处理大规模代理的并行行为和决策，提高模拟的效率和性能。

灵活的场景定制

用户根据需求定制模拟场景，包括代理的个性设置、网络拓扑结构和外部事件干预等，适应多样化的研究需求。

丰富的结果分析

模拟结果以结构化数据格式输出，包括agent_id.json、event.json和model.json等文件，为研究者提供丰富的分析数据。

技术原理

深入了解Casevo的核心技术架构

离散事件模拟

基于离散事件模拟机制，用轮询更新方式管理代理行为和事件调度，确保系统行为的同步和事件的有序安排，适合逐步推进的社会动态模拟场景。

大型语言模型（LLMs）集成

与LLMs的集成，让代理生成自然语言文本，进行类似人类的决策和交流，增强模拟的真实性和复杂性。

链式思维（CoT）

CoT机制支持代理进行多步推理，考虑多个因素后再做出决策，模拟战略行为如规划、谈判和联盟构建等。

检索增强生成（RAG）

RAG记忆系统使代理能回忆过去的互动和决策，基于历史数据生成更细致和上下文相关的行为，模拟人类的长期战略思维和记忆依赖决策。

模块化架构

Casevo采用模块化设计，将模型设置、代理行为定义、并行优化和网络管理等功能划分为独立模块，实现系统的高度灵活性和可扩展性，便于根据具体需求进行定制和扩展。

操作流程

使用Casevo进行模拟实验的完整步骤

1

环境配置与安装

环境要求

确保拥有Python 3.11以上的运行环境。

项目安装

通过pip命令安装Casevo的whl包文件：

pip install casevo-0.3.12-py3-none-any.whl

密钥配置

在代码中配置所使用的大语言模型服务的API密钥。

2

模拟场景构建

定义社会网络

使用networkx等库创建智能体之间的社会关系网络，并保存为JSON数据。

定义智能体画像

创建person.json文件，定义每个智能体的ID、初始观点、角色等属性。

生成主配置文件

将网络结构和智能体画像整合到主配置文件case_lite.json中，作为模拟的输入。

3

运行模拟与结果分析

编写运行脚本

编写run.py，用于加载配置文件、实例化模型并控制模拟的步进。

执行模拟

在命令行中运行脚本，并传入配置文件和模拟步数作为参数：

python run.py case_lite.json 6

分析结果

模拟结果将输出到log文件夹中，包括agent_id.json, event.json和model.json等文件，记录了每个智能体的决策过程和全局事件，为研究者提供丰富的分析数据。
